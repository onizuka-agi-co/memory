## 📋 新規企画

### 🎋 AI研究論文ナレッジグラフ

**概要:**
収集したAGI論文をナレッジグラフとして可視化し、論文間の関連性を探索できるシステム。

**背景:**
papers-collectorで収集した論文は増加しているが、論文間の引用関係やトピックの関連性が見えない。ナレッジグラフ化することで、研究トレンドや技術の発展を視覚的に把握できる。

**機能要件:**
- 論文メタデータの自動抽出（タイトル、著者、引用数、キーワード）
- 論文間の関連性の自動推定（共著者、引用、キーワード共起）
- インタラクティブなグラフ可視化
- タグ・カテゴリによるフィルタリング
- 検索機能

**技術スタック（候補）:**
- グラフDB: Neo4j / ArangoDB
- 可視化: D3.js / vis.js / Cytoscape.js
- ホスティング: GitHub Pages / Vercel

**設定:**
- Priority: P1
- Size: L
- Start Date: 2026-03-21
- Target Date: 2026-03-28
