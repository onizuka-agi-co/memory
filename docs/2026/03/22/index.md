**🔍 AGI知識ベース検索エンジン**

日報・メモリー・タスクを横断検索できる知識ベースを構築。

**背景:**
- 日報: 30日分以上蓄積
- タスク: 30件完了
- 議事録: ミーティング毎に蓄積
- 現状: ファイル単位でしか検索できない

**機能:**
- 🔍 横断検索 - 日報・タスク・議事録を一括検索
- 🏷️ 自動タグ付け - 技術用語・プロジェクト名を自動抽出
- 🔗 関連性表示 - 関連する日報・タスクをリンク
- 📊 可視化 - 知識の関係性をグラフ表示

**技術スタック:**
- Meilisearch / Elasticsearch（全文検索）
- SQLite（メタデータ管理）
- VitePress（フロントエンド）

- Size: L
- Start: 2026-03-23
- Target: 2026-03-30
