- 追加要望に基づき、KPI表へ `新規企画の数（月次）` を追加。
- あわせて、運営改善を追える指標を追加。
  - 企画の検証リードタイム（提案→検証完了）
  - コミュニティ継続参加率（翌月継続）

## 4. 日本語表現の修正
- 不自然な表現を修正。
  - `おし上げる` → `押し上げる`
