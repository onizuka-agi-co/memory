## 🎯 選定アイデア

### AGI論文ウォッチャー

**概要：** 毎週最新AGI論文を1-2本選定し、要約・解説記事を作成。Gemini Visionで図表を解析、nano-banana-2で概念図を生成し、X Communityと連携して議論を促進。

**期間：** 中期（1-3ヶ月）
**優先度：** P2

### 実行計画

1. **Step 1:** arXiv API連携スクリプト作成
2. **Step 2:** LLM要約ワークフロー確立
3. **Step 3:** 週次実行スケジュール（S6サービス）
4. **Step 4:** Communityへの自動投稿

### 技術要件

- arXiv API
- Gemini Vision（図表理解）
- nano-banana-2（概念図生成）
